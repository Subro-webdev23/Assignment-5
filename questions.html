Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer:
innerHTML প্রোপার্টি দিয়ে HTML এর (যেমন div বা p ইত্যাদি) ভিতরের সব contant নিয়ে আসা যায়
innerText প্রোপার্টি দিয়ে কোনো Element ভিতরের লেখা পরিবর্তন করতে ব্যবহার হয়
textContent > text কনটেন্ট রিটার্ন করতে ব্যবহার হয়

Question-3: What is event delegation in the DOM?

Answer:
event delegation হলো এমন একটি কৌশল, যেখানে কোনো event listener একাধিক child এলিমেন্টের সাথে সংযুক্ত করার পরিবর্তে, প্যারেন্ট এলিমেন্টের সাথে সংযুক্ত করা হয়।

Question-4: What is event bubbling in the DOM?

Answer:
event bubbling এর মাধ্যমে, ইভেন্টটি সবচেয়ে ভেতরের লক্ষ্য উপাদান থেকে শুরু হয়ে, তারপর পিতামাতাদের উপর ধারাবাহিকভাবে ট্রিগার হয়।
